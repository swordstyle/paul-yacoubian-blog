# Personal Blog

A minimal, clean personal website built with Next.js and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

Edit `lib/content.ts` to update:
- Your name and bio
- Links to your builds, art, essays
- Book recommendations
- Angel investments
- Missions you care about

## Deploy on Vercel

1. Push to GitHub
2. Connect repository to Vercel
3. Deploy automatically

## Project Structure

- `app/page.tsx` - Main homepage
- `lib/content.ts` - All content configuration
- `app/layout.tsx` - Site layout and metadata
- `app/globals.css` - Tailwind CSS styles
